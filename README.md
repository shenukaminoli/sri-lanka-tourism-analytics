# Sri Lanka Tourism Analytics – Power BI

## Overview

Interactive Power BI dashboard analyzing Sri Lanka tourism trends including arrivals, revenue, hotel occupancy, and seasonality using DAX and visual analytics.

## Features

* KPI cards with trend & target
* Country wise arrivals (map & bar)
* Revenue vs arrivals comparison
* Season analysis (Peak/Off-Peak)
* Purpose of visit distribution
* Interactive slicers

## Dataset

`srilanka_tourism_dataset.csv`
Fields: Year, Month, Country, Arrivals, Revenue_USD, Purpose, Hotel_Occupancy

## Tools

Power BI • DAX • GitHub Desktop

## Key Insights

* Peak season: Dec–Feb
* Holiday travel dominates
* Revenue strongly follows arrivals
* Avg occupancy above 70%
